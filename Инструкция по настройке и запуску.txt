Инструкция по настройке и запуску проекта «Corporate Dinner Dash».

1. Скачиваем и устанавливаем VS Code с сайта: https://code.visualstudio.com/
2. Переходим в "Расширения" (Ctrl + Shift + X) ищем и устанавливаем следующие расширения:
	- Python v2023.16.0
	- Russian Language Pack for Visual Studio Code v1.82.2023091309
3. Перезагружаем VS Code
4. Открываем терминал (PowerShell) от администратора (если нет доступа просим организаторов помочь), пишем и запускаем: Set-ExecutionPolicy RemoteSigned
5. В VS Code открываем новую папку ("Файл > Открыть папку..." или "Ctrl + K Ctrl + O"):
	- В левой панели выбираем "Документы"
	- В верхней панели нажимаем "Новая папка", или в пустой области правой панели нажимаем правую кнопку мыши и выбираем пункт "Создать" > "Папку"
	- Называем папку "cdd_project" и нажимаем "Выбор папки"
	- Копируем содержимое этой папки в "cdd_project"
6. Создаем виртуальную среду Python в VS Code:
	- Нажимаем Ctrl + Shift + P
	- Вводим "Python: создать среду..."
	- Выбираем "Venv", а затем системную среду Python (Если нет системной среды, то скачиваем по ссылке: https://www.python.org/ftp/python/3.11.5/python-3.11.5-amd64.exe и устанавливаем), 
	отмечаем галочкой использование файла requirements.txt для установки дополнительных модулей (Django, Pillow и т.д.)
7. Создаем терминал в VS Code ("Терминал > Создать терминал" или "Ctrl + Sift + `"). Если все правильно сделано, то вторая строчка в терминале будет начинается с "(.venv)", 
если нет, то смотрим пункт 4 и 6, затем обращаемся за помощью к организаторам. (Завершить работу терминала можно кнопкой с изображением мусорного ведра в окне терминала в верхнем правом углу)
8. Запускаем проект командой в терминале "python manage.py runserver" и проверяем работоспособность (переходим в браузере по ссылке http://127.0.0.1:8000/)
9. Для редактирования базы данных переходим в панель администрирования "http://127.0.0.1:8000/admin" логин: "admin", пароль: "admin"
10. Запускаем как Десктопное приложение на полный экран "python desktop_app.py"
